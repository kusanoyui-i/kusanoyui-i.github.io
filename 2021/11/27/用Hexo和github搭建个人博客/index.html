用hexo+github搭建个人博客

发表于2021-11-27|更新于2021-11-27
|阅读量:

注册一个GitHub账号

在GitHub中创建一个仓库，仓库名必须为GitHub用户名.github.io

安装nodejs

点击下载nodejs

下载安装完成后，打开命令行输入npm install -g hexo-cli

等安装完成后再新建一个文件夹，打开命令行进入这个文件夹
输入:
hexo init
npm install

去Hexo中选择一个自己喜欢的主题

在hexo根目录中，修改_config.yml文件中的theme:改成你在hexo中选择的主题的名字

发布到github

先打开cmd进入到hexo根目录
输入hexo g就会在hexo的根目录生成一个叫public的文件夹。
打开浏览器，进入到你github的主页，进入刚才创建的项目中
点击Add file下的Upload files

进入到这个界面
现在你去浏览器搜索你这个仓库的名字就可以进入到你的网页中了

如果上传到github后出现没有加载到样式表的问题

首先进入hexo根目录，打开_config.yml文件，修改
修改完成后打开cmd，进入到hexo根目录，输入hexo clean再输入hexo g之后重新把public文件夹中的内容上传到GitHub中即可
